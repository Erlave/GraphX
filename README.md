


# GraphX

GraphX is a simple Python application for **visualizing and analyzing quadratic functions** using an interactive GUI.  
This project was created as a **school practice assignment** to learn Python, Tkinter, and basic mathematical plotting.

> Built with Python, Tkinter, and Matplotlib.
